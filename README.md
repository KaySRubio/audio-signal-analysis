# audio-signal-analysis
Feb 2026 Repo analyzing sound (bats, chickens, etc) with a combo of productionized code for utils and notebooks


## Setup

```bash
# Option 1: Automated setup (requires Task)
# If you don't have it yet, install Task globally with `pip install go-task-bin`, see https://taskfile.dev for details
task setup
source .venv/bin/activate

# Option 2: Manual setup
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```
